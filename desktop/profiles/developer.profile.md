# Developer Profile

Purpose: configure Liam_OS Desktop for software development.

## Candidate features

- Git.
- VS Code or VSCodium selection.
- Docker/Podman.
- Node.js and pnpm.
- Python.
- Rust.
- Go.
- Java.
- PostgreSQL/MariaDB/Redis optional installers.
- Terminal theme/profile setup.

## Packaging rule

Keep large toolchains optional. Install through profile selection rather than bloating the default system.
