# Creator Profile

Purpose: configure Liam_OS Desktop for designers, UI/UX workers, and graphics creators.

## Candidate features

- Blender.
- Krita.
- GIMP.
- Inkscape.
- Font manager.
- Color picker.
- Screenshot and recording tools.
- Pen tablet support notes.
- Figma web shortcut where appropriate.

## Packaging rule

Prefer open-source tools by default. Use external app setup flows for proprietary services.
