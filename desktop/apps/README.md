# Liam Desktop Apps

Future Liam_OS Desktop applications should live here or in separate app repositories.

Recommended app set:

- Liam Welcome.
- Liam Control Center.
- Liam Gaming Hub.
- Liam Dev Hub.
- Liam Creator Hub.

Recommended implementation stack: Tauri + React, or another lightweight native desktop stack. Avoid Electron unless the added runtime size is acceptable.
