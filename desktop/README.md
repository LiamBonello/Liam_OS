# Liam_OS Desktop

This directory is the planning and future implementation area for the market-facing Liam_OS Desktop distribution.

It is intentionally separate from `core/`, which contains the original custom kernel.

## Recommended first desktop base

- Ubuntu LTS or Debian.
- KDE Plasma.
- Calamares installer.
- Liam-branded first-run setup app.
- Optional gamer/developer/creator profile installers.

## Status

This directory is not yet a buildable OS distribution. It is a structured placeholder so desktop product work does not get mixed into the low-level kernel project.
