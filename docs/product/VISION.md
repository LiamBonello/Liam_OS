# Liam_OS Product Vision

Liam_OS is a desktop operating system project for gamers, developers, and creators who want a polished, customizable, performance-aware environment.

## Product promise

Liam_OS should deliver:

- Smooth visual experience and user-controlled animation intensity.
- Strong default support for gaming workflows.
- First-class developer tooling setup.
- Creator-friendly defaults for UI/UX and graphics work.
- A coherent visual identity instead of a renamed upstream desktop.
- Safe, maintainable updates.

## Two-track model

### Liam_OS Core

The original from-scratch kernel. This is the technical research track and should continue developing kernel fundamentals: process isolation, executable loading, storage, graphics, input, and eventually x86_64.

### Liam_OS Desktop

The future commercial product. For a realistic first release, this should use a Linux base, likely Ubuntu LTS or Debian, with Liam-specific branding, defaults, tooling, and first-run setup.

## Product boundary

Liam_OS Desktop can be original as a product without every component being original. Its originality should come from integration, UX, theme system, default profiles, Liam apps, support model, and distribution experience.
